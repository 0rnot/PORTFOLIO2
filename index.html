INFO // 使い方と仕組み

  1. +銘柄追加で登録。オルカン/Slim S&P500/NASDAQ100は公式基準価額を自動取得し毎日再キャリブレーション。
  2. 推定式: 推定基準価額 = 公式基準価額 × 指数変動率 × 為替変動率。基準値は公式基準価額の算出基準日に日付一致させた終値。
  3. 表示値は「次回基準価額のライブ予測」。証券口座の評価額は前回公式基準価額で固定なので、その日の変動分だけ差が出るのが正常(パネルの公式NAV比がその差)。
  4. 米市場休場日(独立記念日等)は先物も停止するためほぼ動かない。動きの有無はデータの実態そのまま(ダミー無し)。
  5. 数字が動かない時は回線診断。指数データは最大15分程度の遅延あり。
  6. データはブラウザのlocalStorage保存。保存/復元でJSONバックアップ。GitHub Pagesに置けばスマホでも見られる。
※ 表示は推定値であり実際の基準価額・評価額と乖離します。投資判断は自己責任で。データ: Yahoo Finance / 三菱UFJアセットマネジメント / Binance / CNN / ForexFactory / NHK / ExchangeRate-API (無保証・遅延あり)。構成銘柄比率は月報ベースの静的概算(コード内CONSTITUENTSで編集可)。
